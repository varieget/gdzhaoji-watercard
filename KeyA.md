# KeyA 计算方式

由固定前缀 `0x36` 与 UID 加上起始后缀 `0x03` 作为 Sector 0 的 KeyA

下一个 Sector 的后缀需要加 `4`

## 分析

例如，这张水卡的 UID 为 `AABBCCDD`

前缀依旧是 `0x36`，后缀保持从 `0x03` 开始、步进为 `4` 的规律
